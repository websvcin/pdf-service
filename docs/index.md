# PDF Render Service

Generate beautiful PDFs from HTML templates using .NET 8, Handlebars and Playwright.

[Docker Hub](https://hub.docker.com/r/websvcin/pdf-service), [Download Latest](https://websvcin.github.io/pdf-service/releases/latest.zip), [GitHub](https://github.com/websvcin/pdf-service)

---

## What this service does

Convert HTML templates into PDF documents using JSON data.

Perfect for:

• invoices
• reports
• certificates
• statements
• agreements

---

## How it works

1. Send HTML template
2. Send JSON data
3. API merges template
4. Playwright renders PDF

---

## Quick Start

**runcontainer**  
`docker run -p 8080:80 websvcin/pdf-service`

**Swagger UI**: http://localhost:8080/docs
